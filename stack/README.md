# Código-fonte (amostra representativa)

Esta pasta contém **ficheiros reais** do projeto DirCompareAndSync, publicados no repositório de releases para que o GitHub mostre as **linguagens utilizadas** na aplicação.

| Tecnologia | Uso |
|------------|-----|
| **C#** / **.NET 10** | Lógica da aplicação (`DirCompareAndSync.Core`, `DirCompareAndSync.Desktop`) |
| **Avalonia UI** | Interface gráfica Windows (ficheiros `.axaml`) |
| **PowerShell** | Scripts de build, publicação e upload de releases |

O repositório de desenvolvimento completo é privado. Esta amostra é **apenas informativa** — não substitui o código-fonte integral.
